# NYTimes bestseller books browser 
Just for fun and learning.

# Set NYTimes API key
Set your NYTimes API key NYT_API_KEY evironment variable.
Creating .env file at root of backend is supported.

# Start backend and frontend
cd backend  
    npm install  
    npm run start  

cd frontend  
    yarn  
    yarn start
