# NYTimes bestseller books browser 
Just for fun and learning.

# Set NYTimes API key
Set your NYTimes API key backend/config/common.yml.
(Will be moved to environment variables in future commits)

# Start backend and frontend
cd backend  
    npm run start  

cd frontend  
    yarn start
